# Customer Churn Prediction: A Comparative Machine Learning Study

This repository presents a comprehensive analysis and development of Machine Learning models to predict **Customer Churn**. The project explores a wide range of architectures‚Äîfrom classical boosting methods to modern tabular deep learning approaches‚Äîto identify the most effective strategy for identifying customers at risk of leaving a service.

## üöÄ Project Overview

The objective of this project is to build a robust predictive pipeline capable of handling tabular data and identifying patterns associated with customer attrition. We implemented and benchmarked multiple models of different natures to ensure the highest predictive performance.

### Key Results
After extensive testing and evaluation across several architectures, the **XGBoost** model achieved the best overall performance, demonstrating superior capability in capturing complex non-linear relationships within the dataset.

## üìÇ Repository Structure

The project is organized into two main directories:

* **`/models`**: Contains individual notebooks for each model tested, including training, hyperparameter tuning, and performance evaluation.

## üß™ Models Evaluated

To find the best solution, we implemented and compared the following architectures:

* **Boosting Methods:** XGBoost (Best Performer), Gradient Boosting.
* **Neural Networks:** Multi-Layer Perceptron (MLP).
* **Modern Tabular Models:** KAN (Kolmogorov-Arnold Networks), TabKAN, TabPFN, and STab.
